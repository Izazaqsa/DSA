## 🎯 Core Concepts Covered

This repository organizes problems and scripts into major computer science building blocks:

### 📥 1. Linear Data Structures
* **Arrays & Strings:** Manipulating memory blocks, two-pointer approaches, and sliding windows.
* **Linked Lists:** Understanding node connections, pointers, and structural loops.
* **Stacks & Queues:** Masterclass in LIFO (Last In, First Out) and FIFO (First In, First Out) patterns.

### 🌿 2. Non-Linear Data Structures
* **Trees & Binary Search Trees (BST):** Hierarchical data management and recursive traversals.
* **Graphs:** Exploring connections using fundamental traversal paths like BFS (Breadth-First Search) and DFS (Depth-First Search).

### ⚡ 3. Algorithmic Paradigms
* **Sorting & Searching:** Implementing classics like Binary Search, Quick Sort, and Merge Sort.
* **Recursion & Backtracking:** Breaking massive problems down into elegant, repeating sub-problems.
* **Dynamic Programming (DP):** Storing sub-problem answers to avoid computing the same logic twice.
